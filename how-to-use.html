How to use

Keytty

Open Keytty

  • Press ⌘(Command) twice (quickly) to open the Keytty interface.
  • You can type the commands listed below after you open the Keytty interface.

Commands

Simple Move
  • ←(H) - left
  • ↑(K) - up
  • →(L) - right
  • ↓(J) - down
Section Move
  • C - center
  • W - top left
  • E - top right
  • S - bottom left
  • D - bottom right

Actions

  • To use action, Type "A"
Click
  • "," - left click
  • "." - right click
  • ";" - double click
Scroll
  • O - scroll up
  • P - scroll down
  • "[" - scroll left
  • "]" - scroll right
Drag and drop
  • N - drag start
  • M - drop (drag end)
Jump (Multiple monitors)
  • [Spacebar] - Jump to the next screen

Special Modes

- Control Mode

  • Toggle by pressing Control Key (⌃)
  • cursor move 100 distance instead of 10.
  • scroll more than normal.

- Alt Mode

  • Toggle by pressing Alt Key (⌥)
  • cursor move 1 distance instead of 10.
  • scroll less than normal.

- Continuous Commanding Mode

  • Toggle by pressing Quote Key (')
  • Keytty interface opens again after execution.

Advanced Settings (Keytty status bar icon click -> Advanced Settings -> [click to toggle])

- Start with "c" (center)

  • The interface automatically type "c" when it is opened by user.

- Execute without Enter

  • Actions will be executed right away when action commands are typed. (No need to press enter to execute actions)

- "a" not required for action

  • No need press "a" to be able to type actions

- Hide Command suggestions

  • Command suggestions next to the typing interface is hidden.
  • (Only shows when typed wrong commands.)

- Start with "Continuous Commanding"

  • The interface start with automatically in "Continuous Commanding Mode", when it is opened by user.
